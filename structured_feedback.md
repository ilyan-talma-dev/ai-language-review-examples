# Structured Feedback Format

When evaluating AI-generated French text, feedback should follow a clear structure:

1. Output
   The sentence produced by the model.

2. Issue Type
   - literal translation
   - tone mismatch
   - cultural misunderstanding
   - grammar issue

3. Explanation
   Describe why the output is incorrect or weak.

4. Suggested Improvement
   Provide a corrected version or guidance.

Clear structure helps ensure consistent and objective evaluation.
